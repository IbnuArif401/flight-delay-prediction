# flight-delay-prediction
Project  ini bertujuan untuk memprediksikan jadwal penerbangan menggunakan algoritma machine learning berdasarkan data dari kaggle

## Dataset
Dataset yang digunakan mencakup informasi jadwal penerbangan, maskapai, bandara asal, bandara tujuan dan status keterlambatan (delay).

## Struktur  Folder 
'customer-churn-prediction.ipynb' - Notebook proses analisis data pemodelan.
'README.md' - Penjelasan mengenai proyek ini

## Cara Menjalankan Project
1. Clone repository ini
2. Buka file '.ipynb' menggunakan Jupyter Notebook, VS Code, atau Google Colab
3. Jalankan setiap sel kode secara berurutan
